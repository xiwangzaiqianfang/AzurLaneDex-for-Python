# AzurLane-Dex
一个使用 **PySide6** 开发的碧蓝航线本地图鉴管理工具，帮助玩家记录舰船收集进度、科技点加成、舰队科技统计等。支持深色/浅色主题切换，数据本地存储，可导出/导入数据。

# 请注意，本仓库已归档，应用已迁移至 WinUI 3,请前往 WinUI 3 版本仓库获取应用最新版本

## 功能特性
**舰船数据管理**：记录每艘船的基本信息、拥有状态、突破数、誓约、120级、改造等。

**科技点系统**：完整模拟游戏内科技点规则，记录三阶段科技点（获得/满破/120级）及属性加成。

**舰队科技统计**：按阵营汇总科技点，显示全舰队属性加成。

**筛选与搜索**：按阵营、舰种、稀有度、状态筛选，支持舰船名实时搜索。

**多账号支持**：切换不同 JSON 文件，实现多账号数据分离。

**导入/导出**：支持 CSV/Excel 格式，方便批量编辑与备份。

**网络更新**：从仓库拉取最新舰船数据，自动合并用户进度。

**深色/浅色主题**：一键切换，保护眼睛。

**可自定义**：支持手动添加新船，编辑科技点数值。


## 安装与运行
使用 **Release** 版本可以直接进行安装

使用**构建**请按照下方步骤进行
### 环境要求
Python 3.8 或更高版本

PySide6 6.5+
### 1. 克隆或下载项目
```
git clone https://github.com/yourname/AzurLaneDex.git
cd AzurLaneDex
```
### 2. 安装依赖
```
pip install -r requirements.txt
```
依赖清单参见：
```
PySide6>=6.5.0
pandas>=2.0.0
openpyxl>=3.1.0
requests>=2.28.0
```
### 3. 运行程序
```
python main.py
```
首次运行会自动生成示例数据文件 ships.json （如果没有）和示例立绘图片（需自行放置图片到 images/ 文件夹）。、

## 使用指南
### 基本操作
左侧列表显示舰船，点击可查看详情。

右侧可修改拥有状态、突破数、誓约、120级、改造。

顶部筛选栏可组合筛选，搜索框实时过滤舰船名。

点击“舰队科技”查看各阵营科技点总和及全舰队加成。

点击“更多操作”可进行新增舰船、切换账号、导入/导出、网络更新等。

### 数据文件
ships.json：存储所有舰船数据，可手动编辑或通过程序修改。

建议定期备份，或将文件放在云同步文件夹（如 OneDrive）中，实现多设备协同。

### 主题切换
程序默认浅色主题，点击“切换主题”按钮可在深色/浅色间切换，设置自动保存。

### 新增舰船
点击“更多操作” → “新建舰船”。

填写编号（0表示自动分配）、名称、阵营、舰种、稀有度、获取方式等。

在“属性加成”区域填写各属性的获得/120级数值。

在“科技点总和”区域填写三阶段科技点。

勾选适用舰种（默认根据舰种自动选择）。

点击确定保存。
